# Personal Curriculum Vitae Website

This repository contains a personal Curriculum Vitae (CV) website created for an HTML midterm exam project. The website presents personal profile information, education background, technical skills, projects, gallery, and contact information in a clean and responsive web layout.

## Project Overview

This website was built to fulfill the requirements of an HTML midterm exam. It uses HTML, CSS, JavaScript, and Bootstrap to create a simple but professional personal CV website.

## Features

* Responsive one-page CV website
* Personal profile section with photo
* About Me section with self-description
* Technical skills and learning roadmap
* Education background table
* Experience and projects table
* Gallery section with images
* Contact section with external links
* Back to Top button using JavaScript
* Built with Bootstrap for responsive layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Website Sections

1. Home
   Displays the main profile introduction, name, role, and call-to-action buttons.

2. About
   Contains a short personal description and uses HTML text formatting tags.

3. Skills
   Shows technical skills and learning roadmap using unordered and ordered lists.

4. Education
   Displays education background using a table.

5. Experience & Projects
   Displays project experience using a table.

6. Gallery
   Displays several images related to profile, cybersecurity, and networking.

7. Contact
   Provides email and social media links.

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` using a web browser.
4. The website can be viewed locally without additional installation.
